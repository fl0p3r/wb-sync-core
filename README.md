# Universal Sync Engine (Node.js + MongoDB)

Система автоматизации товарного учета и синхронизации данных между внешними API, Google Таблицами и Маркетплейсами.

## 🚀 Основной функционал
- **Синхронизация 30,000+ SKU:** Архитектура рассчитана на высокие нагрузки и большие массивы данных.
- **Интеграция с Google Sheets v4:** Управление ценами и наценками напрямую из облачной таблицы.
- **Batch Processing:** Пакетное обновление данных для обхода лимитов (Rate Limits) маркетплейсов.
- **Telegram Logging:** Мгновенные уведомления о статусе работы системы.

## 🛠 Стек
- Node.js (ESM), MongoDB (Mongoose), Axios, Telegraf, Google-spreadsheet.
